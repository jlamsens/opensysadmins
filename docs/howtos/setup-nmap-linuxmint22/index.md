# Setup nmap op Linux Mint 22

Nmap is een krachtige netwerkverkennings- en beveiligingstool die wordt gebruikt om systemen op een netwerk te ontdekken en te analyseren door open poorten, actieve services, gebruikte protocollen en vaak ook het besturingssysteem van een host te detecteren, wat het onmisbaar maakt voor netwerkbeheer, troubleshooting en beveiligingsaudits.
